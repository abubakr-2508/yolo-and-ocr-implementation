Note: This is a Python desktop application that runs locally with a webcam. It cannot run directly in the browser due to browser security restrictions on camera access and the need for Python backend processing.

Features

  • Real-time object detection with YOLOv8
  • Text recognition with EasyOCR
  • Combined display of detection results and recognized text
  • High-resolution camera feed (1280x720)

How to Run Locally

1. Clone or download this repository:

git clone https://github.com/abubakr-2508/yolo-and-ocr-implementation.git

2. Install the required dependencies:

pip install ultralytics opencv-python easyocr numpy

3. Run the application:

python detect.py

Press 'q' to quit the application.

Web Version (Streamlit)

We've also created a web version using Streamlit that works with image uploads:

pip install streamlit streamlit run app.py

Repository Contents

  • detect.py - Main desktop application with webcam
  • app.py - Streamlit web application version
  • yolov8n.pt - Pre-trained YOLOv8 nano model
  • requirements.txt - Python dependencies
  • README.md - Project documentation
